# University Thesis Management System - Frontend

## UI Updates

The frontend UI has been updated to match the DISI (Decision Information System of IOE) styling with a new color theme:

### New Features:
- Modern, clean interface with academic styling
- Responsive sidebar navigation with Material Symbols icons
- Consistent color scheme using CSS variables
- Enhanced dashboard with bento cards and statistics
- Improved form elements and modal dialogs
- Better typography with Inter and DM Sans fonts
- Material Symbols integration for icons

### Color Theme:
- Primary: Blue (#2563eb) for main actions and highlights
- Secondary: Teal (#0891b2) for supporting elements
- Tertiary: Purple (#7c3aed) for special actions
- Surface: Light gray (#f8fafc) for backgrounds
- Error: Red (#dc2626) for error states
- Success: Green (#16a34a) for success states

### Key Components:
1. **Sidebar Navigation** - Collapsible menu with role-based links
2. **Dashboard** - Overview with statistics cards
3. **Data Tables** - Sortable tables with action buttons
4. **Forms** - Consistent form styling with validation
5. **Modals** - Enhanced dialog windows for CRUD operations
6. **Cards** - Content containers with consistent styling

### Responsive Design:
- Adapts to mobile, tablet, and desktop screens
- Collapsible sidebar on smaller screens
- Flexible grid layouts for content

## Development

To run the application:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```
